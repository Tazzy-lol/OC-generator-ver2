# OC Generator

a python program that lets you create random OCs with different genres, personalities, powers, weapons, and lore.

## features

- fantasy, cyberpunk, and horror genres
- choose your own character traits
- random names
- detailed backstories

## example

```txt
Name: Luna
Genre: Fantasy
Personality: Quiet
Weapon: Sword
Power: Fire magic
```

## built with

- Python
- random module
